# 项目迭代面板（plan.md）

我们用这个文件做轻量迭代看板：每次开始一个新任务就新增一条卡片；实现完成并通过验收后，将卡片移动到 Done，并勾选验收标准。

## 需求卡片模板（复制后填写）

### [ID] 标题
- 用户故事：作为[角色]，我希望[能力]，以便[价值]
- 验收标准：
  - [ ] 条件 1
  - [ ] 条件 2
  - [ ] 条件 3
- 关联（可选）：页面/接口/关键文件
- 记录（可选）：创建日期 YYYY-MM-DD；完成日期 YYYY-MM-DD；备注
